# 台股大迷思：量先價行

> 原文來源：https://www.jgtruestock.com.tw/

## 核心主張

1. **「量先價行」是個假議題** — 市場上普遍流傳的量價法則根本定義不清，「大量」、「窒息量」這些詞都是模糊概念，沒有明確標準。

2. **直接看價格就夠了** — 與其糾結有沒有量、量夠不夠大，不如專注在結果（價格上漲）。學生考第一名，我們何必管他有沒有念書？

3. **量價背離反而是大漲的信號** — 大立光的例子證明：量能越來越少、價格卻越來越高，結果從300噴到1000、從500漲到1500，都漲了3倍。如果依照傳統理論應該不會漲，但事實證明這些理論都錯了。

4. **大量指標會增加判斷難度** — 多個確認指標疊加會產生「雙重確認的困難」，反而害你錯失機會。KISS法則（Keep It Simple Stupid）才是正道。

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **量先價行** | 傳統股市俗語，認為交易量領先價格變動，但 JG 認為這根本是迷思 |
| **大量** | 無法被清楚定義的模糊概念，159 vs 160 vs 175 到底差在哪裡？ |
| **窒息量** | 同樣無法定義的術語，像女友說「隨便」一樣，所有答案都對又都不對 |
| **量價背離** | 價格上漲但交易量縮減的現象，傳統分析認為不健康，但大立光證明這不一定成立 |
| **致命一擊** | JG 的核心心法：在股票戰場中要專注在真正能決勝負的因素，而非繁瑣指標 |
| **KISS 法則** | Keep It Simple Stupid —— 簡單才是王道，不要把交易搞複雜 |

## 實操規則

- **看結果，不看過程**：如果價格一直上漲就買進，別管成交量有沒有配合
- **忽視傳統量價法則**：不要因為「沒補量」或「量價背離」就不敢買，反而可能錯失大漲行情
- **聚焦在一個信號**：不要同時看 KD、均線、三大法人、成交量……一次看太多會眼花
- **量對初心者沒用，但對進階者有秘密**：JG 承認量有「特殊用法」，但那是超進階技巧，需要很多實戰經驗才能領悟（目前不公開）

## 重要比喻或案例

### 1. 「女友身高」的故事
JG 說年輕時堅持未來女友要「夠高」，但朋友追問「160、159、175 到底哪個算高」，才發現根本無法清楚定義。

**類比：** 就像「大量」一樣——誰能精確定義什麼叫大量、什麼叫窒息量？都是模糊不清的主觀標準。

### 2. 學生考試的比喻
「一個學生每次都考第一名，我們何必管他是有念書還是沒念書呢？」

**延伸：** 股票漲就是漲，不用管交易量配不配合，結果才是唯一的評斷標準。

### 3. **大立光（3008）的實例**
- **左邊黃線：** 300 → 1000（漲 3 倍），過程中量能越來越少
- **右邊黃線：** 500 → 1500（漲 3 倍），一樣量能越來越少
- 這兩波都完全違背「量價配合」的傳統教學，卻都大漲 3 倍
- **反問：** 有沒有量、大不大量、補不補量，這真的重要嗎？

## 與其他觀念的連結

### 連結到「反市場邏輯」
- 傳統市場教學說「量縮要小心」，但現實中量縮反而是大漲前的徵兆
- 這呼應了 JG 一貫強調的「推翻錯誤觀念」主題

### 連結到「KISS 原則」
- 不只這篇，JG 多次強調簡化交易系統的重要性
- 太多指標 = 太多噪音 = 錯誤決策增加
- 專注於「一擊必殺」（看價格）比「十八般武藝樣樣精通」更有效

### 連結到「成本與學費」
- JG 坦誠自己花了多年學費才明白這個道理：看量是浪費時間
- 警告新手別被「牛鬼蛇神」的教學所騙，花大錢學錯誤知識反而害人

### 連結到「經驗與領悟」
- JG 透露量其實有「超進階的台股量架構」秘密，但不適合現階段公開
- 強調有些知識不是一時半刻能講清楚，需要親身實戰幾年才能體悟

## 寫在最後——給戰友的話

JG 用很坦率的語氣說：這篇文可能會被誤解成「量一點屁用都沒有」，但其實**不是這樣**。量在台股確實有用，只是那是進階知識。

現階段戰友要做的是：
- 先把基礎打穩（專注看價格）
- 別被複雜指標迷惑
- 隨著經驗累積，自然會懂得量的真正用法
