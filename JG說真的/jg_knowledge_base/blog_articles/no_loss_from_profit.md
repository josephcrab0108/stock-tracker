# 有賺不能賠：台股停利甜區的終極心法

> 原文來源：https://www.jgtruestock.com.tw/

## 核心主張

1. **市場上大多數人賠錢的真實原因不是買點差，而是賣點差** — 輸家總是在虧損時才賣出，贏家則懂得在獲利時停利。反過來看自己的對帳單（把買改放空、賣改買進），你會發現自己其實一直在做反向操作。

2. **沒有停利方法就不應該待在市場** — 停利不是進階技能，而是基本求生技能。在達到「妙算」階段前沒有建立停利紀律，長期下來幾乎不可能賺錢，因為你把命運交給了情緒。

3. **「有賺不能賠」是渡過停利悟道期前的求生法則** — 在還沒抓到停利甜區前，最低限度就是：絕對不要讓一筆賺錢的單子以虧損收場。這個簡單原則可以度過停損地獄。

4. **掌握停利甜區才能實現「精準換股」進而接收暴利** — 停利甜區是根據股票性質、期貨商品特性而定的標準化流程，掌握它能帶來安心感和持續獲利能力。

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **停利甜區** | 根據股票或期貨商品特性訂定的標準停利點位，例如期貨極短線約6點、當沖長單不超過70點；股票則因股性而異 |
| **妙算** | 來自《操盤階梯》第五階，指建立適合的停損比例和停利方式，是穩定獲利的關鍵步驟 |
| **停損地獄** | 陷入頻繁停損、無限循環的困境，反而因為太常止損而錯過行情 |
| **選擇性停損** | 只在特定高風險情況下停損，而不是機械式地照單全收，這背後的意義是「賣點比買點重要」 |
| **精準換股** | 掌握停利甜區後能夠精確判斷何時出場、何時進場的操作方式 |
| **有賺不能賠** | JG的核心心法，意思是即使虧損也不能讓已經獲利的單子最終以虧損收場 |
| **崩潰線型** | 文中提到的一種技術型態，適合作為買點參考 |

## 實操規則

### 最基本的求生法則（給還沒領悟停利甜區的人）

**JG 心法：「絕對不要讓賺錢的單子以賠錢出場。」**

具體執行方式：
- **期貨**：賺 50 點以上絕對不能虧損收場
- **股票**：賺了 10% 以上，最差最差也要在 1% 時出場

### 實際操作案例（以崩潰線型為例）

情境：你在 7/25 尾盤或 7/28 早盤買進，買點漂亮

| 時間點 | 走勢 | 應該做什麼 |
|--------|------|---------|
| 隔日開高 | 股價上漲 | 這是短線最大獎勵，適合停利點 |
| 開高後回軟 | 股價漲幅縮小 | 保持警覺 |
| 一路回落到成本 | 接近買價 | **快速停利出場**，即使只賺一點也要出場 |
| 之後 | 等待下次機會 | 精準換股，等待下一個買點 |

### 判斷的核心邏輯

> 買點再漂亮，賣點差也是賠錢  
> 買點差，賣的漂亮卻常常可以化解危機

**這個不對等性是整個遊戲的關鍵。**

## 重要比喻或案例

### 1. 反向對帳單測試
JG 教大家的「驚人發現」實驗：
- 把對帳單中所有買進改成放空
- 把所有賣出改成買進
- 結果：90% 的人竟然變成贏家，原本賠越多的績效越好

**含義**：你不是沒有能力，而是方向完全反了。這說明了絕大多數散戶的問題出在「賣的時機」。

### 2. 百看不厭的自殺法（死亡迴圈）

典型散戶故事：
1. 信心滿滿買進 → 2. 看到微小漲幅就不滿足 → 3. 期待漲 10%、30% 甚至賺一半 → 4. 加碼求更大獲利 → 5. 大殺盤來臨 → 6. 慘賠後賣出 → 7. 追高買回 → 回到第 2 步

**JG 的評價**：「無限循環這個劇本」 — 這就是停損地獄。

### 3. 門徒案例（堂弟 1 號）
- 年輕人天生愛錢，一聽到「停利甜區」就領悟
- 因此績效一直維持高檔
- 但後來年初績效太好反而全融資  
- 結果變成：年紀輕輕卻在挑戰第六階段的風險

**隱含意義**：停利能力強≠風險管理能力強，融資槓桿會放大一切。

### 4. JG 個人轉折點
- 2011 年之前：已專職 3 年但賺得辛苦
- 掌握停利甜區之後：突然有「莫名的安心感」
- 正是這份安定感讓他敢於出走台灣

**含義**：停利甜區不只是賺錢工具，更是心理建設和自由的基礎。

- **第五階「妙算」**是核心關鍵 — 必須建立標準化的停損比例和停利方式

- 妙算後的訓練會讓原本「很爛的技術」開始賺錢、讓經典書籍的邏輯浮現價值
