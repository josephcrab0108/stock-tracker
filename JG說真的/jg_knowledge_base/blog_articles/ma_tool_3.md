# 我的關鍵工具 移動平均線(三) 均線操作在台股的特殊意義

> 原文來源：https://www.jgtruestock.com.tw/%e6%9c%80%e6%96%b0%e6%96%87%e7%ab%a0/%e6%88%91%e7%9a%84%e9%97%9c%e9%8d%b5%e5%b7%a5%e5%85%b7-%e7%a7%bb%e5%8b%95%e5%b9%b3%e5%9d%87%e7%b7%9a-%e4%b8%89/

## 核心主張

1. **不要追求「萬用技術分析」** — 很多交易人窮其一生找「對所有情況都有效」的方法，結果就像訂製「什麼場合都能穿」的西裝，最後只會「四不像」。交易工具應該因時制宜，根據市場情況調整，而不是企圖找一招鮮吃遍天。

2. **決勝點在於「觀察大盤」，而非個股技術** — 台灣所有個股的走勢都只是大盤日線的微小變型。把大盤吃透徹，你就能減少複雜的交易邏輯，從十幾種簡化到只需要兩三種。真正的效率來自於大方向的判斷，而不是更複雜的技術分析。

3. **均線的核心是「方向」而非「突破點」** — 看月線角度是否往上揚，不是看股價有沒有突破月線。找到方向後，用5根日K的時間跨度作為出場判斷（跌破月線5天後才考慮賣出），這是台股特性決定的。

4. **成功的交易人應該有生活品質** — 整天鑽研無謂的技術分析，導致績效不穩定、身體僵硬、壓力極大，這不算真的贏。所以問題不在於能否賺錢，而在於是否找到了正確的思考方向。

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **20MA（黃金週期）** | 台股20根日K移動平均線，反映個股的基本趨勢方向，是判斷大多數台股個股的關鍵周期 |
| **月線角度** | 不看股價是否突破月線，而是看月線本身的走勢方向（往上揚或往下跌），這才是買賣的信號 |
| **5根日K法則** | 在台股，5根日K足以改變20MA的趨勢方向，所以跌破月線5天後才考慮賣出 |
| **三大交易邏輯** | 根據大盤方向分為「多頭噴出」「空頭下殺」「多空不明區」三類，每類有截然不同的操作方式 |
| **大盤就是方向** | 就像旅行先定東西南北，大盤決定了市場方向，個股特性（快慢不同）是在這個方向下的變化 |
| **均線解機率，型態解原理** | 均線可解決60%的問題（治標），加上型態可解80%（治本），但根本關鍵在5分K的情緒變化 |

## 實操規則

### 買進條件
- **第一步**：觀察月線角度是否往上走揚（不看股價位置，只看均線方向）
- **第二步**：月線翻揚後盡快買進
- **心態**：一旦開始買進，就要有「抱到天荒地老」的準備

### 賣出條件
- **唯一出場點**：跌破月線5天後才可以考慮賣出
- **重點**：在此之前絕對不能有一絲絲賣出的念頭，否則不只少賺，還會賠掉「生命」（指專職交易最寶貴的時間資源）

### 選股與複製
- **簡單法**：找1000多檔台股中和2454類似的「孿生兄弟」，重複同樣的操作邏輯
- **高效法**：深入研究大盤型態，順著大盤方向找股票，自然會找到合適的標的

### 分辨市場狀態的能力
- 不一定要找「在所有情況都有效」的方法
- 重點是分辨「現在是哪種情況」（左邊/右邊？多頭/空頭/混沌區？）
- 能正確分類市場狀態，就能通吃80%以上的機會

## 重要比喻或案例

### 1. **西裝比喻**（最核心的類比）
JG訂製西裝時提了五個「聰明要求」：正式、可搭牛仔褲、夏天不熱、冬天擋風、不像上班族。裁縫師冷冷地說：「你以為在演不可能任務嗎？」

**啟示**：西裝有宴會服、上班服、休閒服的區分，因應不同場合需要不同布料和織法。交易也是如此——不能期待一套方法對所有市場狀態都有效。這正是為什麼那些老手不斷微調參數（20MA→21MA→25MA），十幾年後還在罵技術分析沒用。

### 2. **旅行比喻**
旅行的三步驟：
- 第一步：確定東西南北的方向 → 大盤方向
- 第二步：順著方向選交通工具 → 選擇適合的個股（有的快如飛機，有的慢如慢跑鞋）
- 第三步：用工具到達目的地 → 執行交易賺錢

**啟示**：大多數交易人倒著做，先糾結個股選擇和技術細節，卻沒搞清楚大方向。JG反過來：大盤方向對了，選股就容易了。

### 3. **聯發科2454案例**
以2005年的聯發科為例，用單純的20MA，該如何最多地賺到錢？
- 不可能做到每個綠K賣出、每個紅K買進
- 真正的做法是盯著月線角度，翻揚就買，跌破月線5天後才賣
- 這個方法簡單但需要紀律，不能中途放棄
