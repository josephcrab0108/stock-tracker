# JG的關鍵心法 賭場原理(一)

> 原文來源：https://www.jgtruestock.com.tw/%e6%9c%80%e6%96%b0%e6%96%87%e7%ab%a0/jg%e7%9a%84%e9%97%9c%e9%8d%b5%e5%bf%83%e6%b3%95-%e8%b3%ad%e5%a0%b4%e5%8e%9f%e7%90%86%e4%b8%80/

## 核心主張

1. **股市本質是「很認真的賭下去」**
   - 台股二十年來，純投資機會不到兩年
   - 必須承認市場的隨機性和運氣成分，而不是盲目相信基本面

2. **「賺賠比」比「勝率」更重要**
   - 你不需要預測對才能賺錢，只要賺賠比夠高、勝率夠低就能獲利
   - 例如 5:1 的賺賠比，只需要 20% 的勝率（十次交易贏兩次）就能賺錢

3. **選對時機比選對標的更關鍵**
   - 即使基本面再好的公司，買進時機錯誤也會長期虧損
   - 涵嘉的故事就是最好的反面教材

4. **台股環境特殊，西方理論要調整**
   - 台股主力複雜（外資、在地幫對立）、股性多變、「週期」影響大
   - 書上的 50% 勝率法則在台股可能根本不適用

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **賺賠比** | 平均獲利金額 ÷ 平均虧損金額的比值，是衡量交易模式是否可行的核心指標 |
| **期望值** | 專業賭徒最重視的東西，簡言之就是把賺賠比拉高的數學模型 |
| **股性** | 每檔股票獨特的價格波動特性和主力操作慣性（下篇文章待解） |
| **週期** | 市場的時間週期特性，影響買賣時機的選擇（下篇文章待解） |
| **技術分析** | 觀察 K 線圖、找出蛛絲馬跡的方法，因為 K 線對所有人都公平 |
| **基本面分析** | 看財報、產業、業績，在台股容易失效因為時機選擇最難 |

## 實操規則

### 📊 計算賺賠比的三步驟

假設用 KD 指標交易 0050：

**第一步：回測找出平均獲利**
- 查看歷史上，每次 KD 低檔向上交叉買進、高檔向下交叉賣出能賺多少
- 假設回測結果：平均每次賺 5 萬

**第二步：設定停損點**
- 根據獲利預估來反推停損
- 設停損為 1 萬，則賺賠比 = 5:1

**第三步：計算所需勝率**
- 進場 10 次，需要贏幾次才能獲利？
- 只要 2 次（20% 勝率）就夠了！

### ✅ 交易前必問的兩個問題

在買進前必須清楚知道：
1. **最多會賠多少？** → 停損點
2. **可能賺多少？** → 預期目標

如果答不出來，先停下來鑽研，不要盲目進場。

### 💡 JG 本身的交易數據參考
- 勝率：60% 多一點
- 賺賠比：3:1
- 評估：足以達到財務自由（勝率和高賺賠比缺一不可）

## 重要比喻或案例

### 📖 案例一：涵嘉的悲劇

涵嘉是會計雙主修、研究基本面十多年的強人，卻遇到：
- ❌ 業績好、產業強、財報沒問題 → 股價還是一路跌
- ❌ 因為基本面沒問題而一直攤平 → 虧損越來越深
- ❌ 終於受不了賣出後 → 股價開始狂漲

**痛點**：最痛苦的不是賠錢，而是「我這麼認真為什麼市場要這樣對待我」

**啟示**：基本面很重要，但 **時機選擇才是決定賺賠的關鍵**

---

### 📖 案例二：彼得林區的智慧

美股投資前輩（彼得林區）的做法和涵嘉看的東西一樣，但他多做了一件事：

**買進時機的講究**
- 年底節稅前的賣壓行情買進
- 大跌時採用「崩盤 All In 大法」積極買進

**賣出時機因公司類型而異**
- 快速成長型 → 等盈餘開始縮水或營運不再擴大時賣出
- 穩定成長型 → 股價漲幅超過盈餘成長幅度時賣出，再等股價下跌才買回

**啟示**：同樣分析能力，加上「時機選擇」才是贏家
