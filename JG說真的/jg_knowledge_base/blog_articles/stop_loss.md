## 核心主張

1. **停損是投資經典的陷阱**
   - 照本宣科地套用國外交易書籍的停損觀念在台股會虧損，因為台股有國外市場沒有的特性（特別是跳空）

2. **不同階段要用不同策略**
   - 新手應該「不停損」+ 慎選戰場和時機
   - 進階者應該學會「選擇性停損」而非死板停損
   - 贏家靠的是判斷力，不是紀律

3. **虧損不是失敗，而是學習的一部分**
   - JG 自己在 2006 年因為狂砍停損而在多頭行情中斷手斷腳
   - 凹單的朋友也虧，死板停損的他也虧——都是輸家，只是輸的方式不同

4. **操盤的藝術在於「妙算」階段**
   - 真正能賺錢的技巧是判斷何時該等、何時該砍、何時該加碼
   - 這不是程式交易能做到的，必須累積經驗和技術分析能力

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **選擇性停損** | 不是盲目停損，而是根據隱形支撐、技術面判斷後，有意識地決定要不要停損 |
| **隱形支撐** | 市場上漲波段中的多個支撐點，需要加強技術分析才能發現 |
| **跳空風險** | 台股特有現象，停損價被一口氣跳過，此時不能機械式停損 |
| **凹單** | 看好後市持續加碼攤平，贏家做這叫「選擇性停損」，輸家做這叫「凹單」（差別在判斷力） |
| **妙算** | 操盤階梯第五階段，能判斷加碼、攤平、止損時機的高級技巧 |
| **賺賠比** | 風險報酬比，高勝率策略的賺賠比可能只有 1:1.7，但能穩定獲利 |

## 實操規則

### 當跳空超過停損價時的決策樹

```
價格跳空超過停損點
    ↓
虧損金額過大？
├─ 是 → 「先砍再說」（情緒會影響判斷）
└─ 否 → 往下
    ↓
檢查隱形支撐 A、B、C...
    ↓
能判斷後續是否反彈？
├─ 不能判斷 → 砍出場
└─ 能判斷 → 「選擇性停損」
    - 不停損，靜觀其變
    - 觀察支撐是否有效
    - 有效就會反彈
```

### 新手不停損戰術（最實用的建議）

**操作規則：**
- 進場條件：台股 6000 點以下
- 操作方式：越跌越買，分批買進 0050
- 出場方式：拉到 7800 點，每上漲 150 點分批賣出
- 成果預估：200 萬投入，五年內可賺 100 萬以上（賺賠比約 1:1.7，勝率極高）

**為什麼這樣做有效？**
- 不用選股（直接買大盤 0050）
- 不用精準進場（用區間判斷，不用時間點）
- 高勝率（只要不在高點進場，大多會賺）

## 重要比喻或案例

### 案例 1：JG 的 2006 年血淚史
- 嚴格停損 → 不停損朋友先贏
- 但 06 年五月跌勢來時，停損救了他，凹單朋友慘重
- **結論**：看起來他贏了，但回頭算帳，多頭行情中他斷手斷腳，整波沒賺到翻身錢——其實也是輸家

### 案例 2：2014 年 7 月 18 日操作
- **背景**：美股前晚暴跌（ADR 收黑），預期台股開低
- **盤前準備**：標出紫色趨勢線和新支撐點 9340
- **早盤**：確實開低破趨勢線
- **正確動作**：不停損，靜觀破線後是否反彈
- **結果**：馬上拉抬，一直強勢到尾盤
- **教訓**：如果死板停損，就在反彈前砍出場，損失獲利機會

### 比喻 1：經典書籍的侷限
> 「照本宣科的應用就像是對著撒旦狂唸金剛經想驅魔一樣，死的不明不白」
- 西方交易書籍 ≠ 台股特性
- 國外商品連續性高、跳空少，所以書上很少提怎麼應對跳空
- 直接套用就像拿錯工具，一樣會失敗

## 與其他觀念的連結

### 與「操盤階梯」的關係
- **階段 1～4（入魔前）**：應該「不停損」為主，不應該頻繁停損
  - 原因：新手接收錯誤技巧，有停損反而是停損地獄
  - 遵守紀律虧的錢 > 不遵守紀律虧的錢
  
- **階段 5（妙算）**：才開始學「選擇性停損」
  - 需要扎實的技術分析基礎（隱形支撐、趨勢線、崩潰線型）
  - 不能死背方法，要深入理解才能應用到新情況

### 與市場心理的連結
