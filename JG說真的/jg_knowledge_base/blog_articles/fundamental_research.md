# 如何研究公司基本面？尋找暴賺潛力公司的觀察重點

> 原文來源：https://www.jgtruestock.com.tw/

## 核心主張

1. **基本面研究的目的不是預測股價，而是找到有「2-3年以上成長動能」的轉機股和快速成長股**
   - JG 的操作邏輯是用總經決定積極或防守策略，再用基本面篩選具體標的

2. **公司基本面的5件事缺一不可，要持續確認公司有沒有實踐目標**
   - 這 5 件事就像是檢查清單，不是一次檢查就夠，要「不停地」追蹤

3. **進場時間點由技術分析決定，基本面只是條件之一**
   - 即使看好的公司，也要等反市場訊號或轉強訊號才重倉，其他時間靠分批進場

4. **致富的人有兩個共通點：成本很低、長期持有，股票至少要拿半年到一年**
   - 暴賺不是靠頻繁交易，而是靠找對公司後的耐心等待

---

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **轉機股** | 公司商業模式改變、從衰退轉向成長的股票，如 MARA 從專利收購轉為挖礦 |
| **快速成長股** | 已經在高速成長軌道上的公司，有明確的擴張計畫 |
| **領先技術** | 不是「現在」產業第一（如台積電），而是「未來」找到新利基市場的技術，通常還在燒錢階段 |
| **資本支出** | 置產、擴廠、設備更新等投資，代表管理層對未來有明確想像 |
| **籌資能力** | 公司能否順利募集資金來支撐成長計畫，金額越大、機會越大 |
| **執行力** | 公司有沒有真的做到說好的事，取決於管理層背景和過去成績 |
| **大局觀** | 總經→操作策略方向 + 基本面→具體標的選擇 + 技術分析→進場時機 |

## 實操規則

### 5 件事檢查清單（持續追蹤）

**① 領先技術**
- 尋找「正在研究未來技術」的公司（雖然股價被低估、風險高，但成功時會爆炸性成長）
- ❌ 別找已經是產業第一的公司（股價已趨於穩定，很難大漲）
- 例子：SENS 開發新的血糖監測產品

**② 轉型計畫**
- 確認公司有清晰的轉型目標
- 轉型成功 = 轉機股變成快速成長股 = 股價倍數漲幅
- 例子：MARA 從專利公司→挖礦公司（3→83 元）

**③ 擴大資本支出**
- 看公司有沒有投入置產、擴廠、更新設備
- 這代表管理層看的不是半年一年，而是更遠的未來
- 💡 財務報表上會顯示開銷增加、負債增加，**但這是好現象**

**④ 籌資能力**
- 觀察公司籌資是否順利、來源是否穩定
- 籌資金額越大 → 轉變機會越大
- 例子：NIO 獲得安徽合肥 70 億人民幣投資，這是轉型成功的關鍵

**⑤ 確實執行**
- 追蹤進度：有沒有按照計畫執行？成果是否符合預期？
- 分析原因：不達預期時，是外部環境（疫情）還是策略問題（銷售無效）？
- 了解管理層：背景、過去成績 → 判斷掌舵能力

### 進場規則

```
基本面看好 ≠ 馬上買進

要等待以下訊號才重倉：
• 反市場技術分析的關鍵點
• 技術面轉強訊號
• 其他時間用分批策略小額進場
```

### 持有週期

- **最少半年到一年**（股票已經是週期相對短的資產了）
- 低成本 + 長期持有 = 倍數成長的致富模式

## 重要比喻或案例

### 案例 1：MARA（美國挖礦公司）
**時間線：**
- 2010 年：專利收購公司
- 2011 年：探勘鈾礦
- 2017 年：轉向比特幣挖礦
- 2020-2021 年：隨著比特幣飆漲，股價從 3 元→83 元

**啟示：** 轉機股最賺的時候，就是產業動能爆發、公司管理層 3-4 年前的佈局開花結果的時刻。

### 案例 2：SENS（血糖監測公司）
**概念：** 針對糖尿病患的新型監測產品 = 領先技術
**特點：** 還在研發階段（容易被低估），但一旦成功就會在市場上領先

### 案例 3：NIO（電動車公司）
**轉機點：** 面臨資金困境時，地方政府安徽合肥投入 70 億人民幣
**啟示：** 政府等大金主願意進來投資 = 公司轉型有希望、外部條件支持

### 比喻：台積電不適合找
- 台積電現在已經是「現在的產業第一」
- 想再有領先突破已經困難 → 股價趨於穩定 → 很難有大幅成長
- **反對比：** JG 要找的是「未來的領先者」，現在還在燒錢的階段

## 與其他觀念的連結

### 與「總經大局觀」的關係
```
總體經濟 → 決定策略方向（積極 or 防守）
        ↓
   基本面分析 → 篩選具體標的（轉機股、快速成長股）
        ↓
   技術分析 → 決定進場時機（最關鍵）
```
