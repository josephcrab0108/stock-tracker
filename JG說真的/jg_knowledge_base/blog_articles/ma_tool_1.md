## 核心主張

1. **均線不是「平均成本」** — 這是市場上最大的誤解。均線只用收盤價計算，無法代表真實的人氣分布（開盤可能一萬人進場，收盤只剩一人，卻用那一人的價格平均），所以葛蘭碧八大法則和「支撐壓力」的邏輯根本站不住腳。

2. **跟著傳統均線訊號進出會賠錢** — JG 親身體驗：聯發科大漲行情，他按照教科書的 20MA 買賣訊號做了 20 次以上的交易，最後還是小賠。反而聽新聞買進的散戶們賺到了。問題不在選股，而在用法錯了。

3. **均線的真正用途是看「方向」，不是看「支撐壓力」** — 把均線想成學生成績的移動平均，我們不會因為一次考試低於平均分就看空，而是冷靜地觀察整體趨勢方向。均線提供的是客觀的多空指向性。

4. **市場上大多數「專家」根本不懂** — 電視名嘴催眠「季線保衛戰」，長期待號子的老散戶被當權威，實際上他們只會看表象。真正能穩定獲利的人用的技術分析方法完全不同。

## 關鍵概念

| 概念 | 解釋 |
|------|------|
| **均線的盲點** | 均線只用收盤價計算，無法反映當天真實的人氣分布，因此「平均成本」的假設本身就是錯的 |
| **葛蘭碧八大法則** | 傳統的均線操作法則（如突破買進、跌破賣出等），但因為均線不代表平均成本，這套法則就變得不可靠 |
| **支撐壓力的幻覺** | 均線有時看起來有支撐力道，但那只是「時靈時不靈的雜訊自我實現」，不是真實的市場現象 |
| **均線的正確用法** | 把它當成一個客觀的「方向指標」，看的是長期趨勢的多空指向性，而不是短期的進出訊號 |
| **多空指向性** | 均線能清楚告訴你市場是往上還是往下，這是它最強大的地方 |
| **頻繁交易的陷阱** | 依賴傳統均線訊號會導致過度交易，手續費和滑價會吃掉所有獲利 |

## 實操規則

### 均線不應該這樣用：
- ❌ 均線往下就賣出，往上就買進（頻繁交易會被吃掉）
- ❌ 跌破均線就認為已經走空（短期波動無法判斷趨勢）
- ❌ 把均線當成成本線或支撐壓力來依賴

### 均線應該這樣看：
- ✅ 把均線當成「方向指標」，冷靜觀察它的長期走向
- ✅ 看的是「多頭還是空頭」的大方向，而不是每天的漲跌
- ✅ 不帶感情地、緊緊盯著均線的方向

### 題目設定的思考：
> 只給你一根 20MA，怎麼靠它賺錢？怎麼賺最多？

JG 故意留下懸念，暗示：
- 不同的均線週期有不同的用途（為什麼是 20MA 而不是 5MA、60MA？）
- 真正會賺錢的技術分析，必須結合「下注原理」才行（這是後續文章會講的）

## 重要比喻或案例

### 1. 學生成績的故事 ⭐
想像你是老師，要用「移動平均線」追蹤某學生的成績：

- 一次考試低於平均分 → 不代表他走下坡（不要看空）
- 一周起起伏伏 → 不代表要不斷改變判斷（不要頻繁買賣）
- 平均分上下徘徊 → 不代表技術分析無用（不要放棄）
- **該做的只有一件事**：冷靜地觀察他的**整體方向**在哪

→ 均線就是這樣。它的價值在於告訴你「這個孩子（市場）目前的趨勢方向」，而不是用來預測每次的小波動。

### 2. 聯發科（2454）的痛苦經驗
- **時間**：2005年，聯發科當時的當紅炸子雞
- **做法**：買技術分析教科書，跟著均線訊號做 20 次以上交易
- **結果**：五個月內聯發科大漲，自己卻小賠出場
- **對比**：只有聽新聞買進的散戶們反而賺到了

→ 這個例子說明：邏輯再縝密、紀律再好，**前提錯了就會慢慢賠光**。而傳統均線用法的前提（「均線 = 平均成本」）就是錯的。

### 3. 均線不等於平均成本的具體數字
假設某一天（如 5/13）：
- **開盤 8893**：有一萬人進場
- **收盤 8808**：只有一人買進
- **均線計算**：只用 8808 這一個價格來平均

→ 問題來了：一萬人的開盤成本能用一人的收盤價代表嗎？顯然不能。
