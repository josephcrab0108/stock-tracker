# 總體經濟對投資有什麼幫助？

> 原文來源：https://www.jgtruestock.com.tw/%e8%82%a1%e5%b8%82%e5%85%a5%e9%96%80/%e5%a4%a7%e5%b1%80%e8%a7%80/%e7%b8%bd%e9%ab%94%e7%b6%93%e6%bf%9f%e5%b0%8d%e6%8a%95%e8%b3%87%e6%9c%89%e4%bb%80%e9%ba%bc%e5%b9%ab%e5%8a%a9%ef%bc%9f%e7%94%a8%e6%8a%80%e8%a1%93%e5%88%86%e6%9e%90%e9%85%8d%e5%90%88%e5%a4%a7%e5%b1%80/

## 核心主張

1. **技術分析需要配大局觀才能追求暴賺** — 單看K棒無法判斷股票漲幅空間（30%vs 1倍），很容易被短期消息干擾而錯過暴賺機會

2. **大局觀 = 總體經濟 + 產業趨勢** — 了解景氣循環位置，能把媒體消息分類成「雜訊」或「警訊」，做到看對、下大、抱住

3. **央行動向是金錢流向的關鍵** — 央行有三大施政目標，通過觀察相關經濟數據（失業率、CPI、PPI）能推測央行政策，進而預判中長期股市方向

4. **景氣循環有脈絡可循** — 四個階段（衰退→復甦→成長→繁榮）會按序出現，每個階段央行政策與市場表現都有對應特徵，可成為操盤的框架

## 關鍵概念

| 概念 | 說明 |
|------|------|
| **大局觀** | 結合總體經濟與產業趨勢的宏觀視角，幫助過濾雜訊、識別警訊 |
| **景氣循環** | 經濟依序經歷衰退、復甦、成長、繁榮四個階段，各階段有特定特徵與政策對應 |
| **央行三大施政目標** | 維持可接受的通膨率、穩定的經濟成長、利率與匯率穩定 |
| **貨幣政策** | 央行透過升息/降息、寬鬆/緊縮來引導金錢流向與市場情緒 |
| **資金+心理** | 柯斯托蘭尼的名言，說明股市由錢的流向與市場情緒共同驅動 |
| **暴賺** | JG的目標風格，不是一夜致富，而是尋找漲1倍以上的股票機會 |

## 實操規則

**觀察這些經濟指標來判斷央行動作：**
- 初請失業救濟金人數
- CPI物價指數
- 生產者物價指數（PPI）

**根據景氣循環階段調整策略：**

| 階段 | 央行態度 | 市場特徵 | 操盤策略 |
|------|---------|---------|---------|
| **衰退期** | 寬鬆救市 | 景氣降溫、股市崩盤、情緒悲觀 | 等待落底信號 |
| **復甦期** | 持續寬鬆 | 油價低、原物料低、經濟數據低基期 | 積極建倉，預期升息前的回檔 |
| **成長期** | 保持寬鬆（不過早緊縮） | 基期走高、信心不足、政策討論激烈 | 留意經濟不確定性影響，準備可能的回檔 |
| **繁榮期** | 緊縮抑制過熱 | 樂觀情緒濃厚、消費投資信心強 | 警惕升息對股市的負面影響 |

**決策邏輯：**
- 優先理解「現在處於景氣循環的哪個階段」，而不是過度解讀個別數據
- 用大局觀過濾消息，區分雜訊（短期波動）vs 警訊（真實趨勢改變）
- 技術分析 + 大局觀 = 看對方向 + 敢下重注 + 能抱住獲利

## 重要比喻或案例

**案例1：2020年疫情衰退期**
- 政府不得已祭出前所未有的救市措施
- FED採寬鬆政策，美元走貶
- JG據此觀察到弱勢美元環境，進而關注比特幣機會

**案例2：2021年復甦期到成長期轉折**
- 2020年11月：市場在復甦期寬鬆環境中，美元持續貶值
- 2021年3月：通膨消息出現，聯準會開始討論升息
- 現象：同樣是多頭環境，但升息預期帶來股市回檔
- 教訓：不能只看「景氣向上」就一路買，要為央行政策轉向做準備

## 與其他觀念的連結

**✓ 與「看對、下大、抱住」的連結**
- 看對 = 用大局觀判斷方向對不對
- 下大 = 有確信的大局觀才敢重倉
- 抱住 = 大局觀幫你區分短期波動（能忍）vs 真實風險（要出場）

**✓ 與「技術分析信念」的連結**
- JG相信K棒反映所有資訊，但K棒無法告訴你「漲幅空間多大」
- 大局觀補足了技術分析的這個盲點
- 兩者結合 = 完整的操盤邏輯

**✓ 與「暴賺心態」的連結**
- 暴賺的前提是找到「能漲1倍以上」的股票
- 這不是單靠技術分析能判斷的
- 需要知道產業在景氣循環哪個位置、央行政策會如何引導資金流向

**✓ 與「雜訊vs警訊」的區分**
- 雜訊 = 景氣向上時的正常回檔，不用理會
- 警訊 = 景氣循環要轉折、央行政策要轉向
