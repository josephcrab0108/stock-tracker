## 一、選股三層架構

```
【第一層】總體經濟大局觀
  → 決定現在應該積極還是防守
  → 看資金往哪裡走（產業比產業、國家比國家）

【第二層】基本面篩選口袋名單
  → 只挑兩種股票：轉機股 or 快速成長股
  → 轉機股兩大標準：
      ① 會不會倒（財務能撐過轉型期）
      ② 對未來的掌控度有沒有在改善（技術護城河/政策支持/人才集中地）
  → 5 件事持續追蹤清單：
      ① 領先技術（未來的第一，現在還在燒錢）
      ② 轉型計畫（有沒有清楚的目標）
      ③ 擴大資本支出（負債增加 = 好現象）
      ④ 籌資能力（大金主願意進來）
      ⑤ 確實執行（管理層有沒有做到說的）
  → 排除條件：股價低於 60 元、走勢像 5 分 K

【第三層】技術面進場
  → 技術信號先觸發，才去深入驗證基本面
  → 核心買點：XHS 急殺（X）之後的情緒恐慌低點
  → 破三（破季線/破段支撐/破大量）配合 XHS 時間
  → 強弱比對：個股 vs 大盤斜率，強勢才考慮
```

## 二、三個操作時期與對應邏輯

### 建倉期
- **前提**：技術面確認急殺過頭、基本面論點成立
- **進場方式**：分批，第一筆不求完美，虧損但看法不變就等下一個信號加碼
- **回檔深度**：3x～5x ATR（急殺過頭才有期望值）
- **速度判斷**：單日大幅急殺比總回檔幅度更重要——速度代表情緒失控

### 加碼期
- **前提**：已有浮盈（股價已漲 ≥ 50%），確認持有論點不變
- **規則**：有賺不能賠——加碼後若部位回到成本，必須全部出場
- **回檔深度**：約 2x ATR（有浮盈墊底，不需要等太深）
- **心態**：用浮盈去賭，本金不動

### 出場期
- **信號**：大盤 vs 個股出現「開口」（走勢背離）兩週以上
- **方式**：策略性減碼（先賣 30%），確認弱勢後繼續減
- **換股邏輯**：賺到的八成當新股成本（`180 - 80×0.8 = 116`）
- **停利哲學**：不設固定停利目標，讓股票漲，在強弱比對轉弱時才出

## 三、速度的角色（關鍵補充）

速度在 JG 系統裡有兩個完全相反的意義：

| 情境 | 速度含義 | 應對 |
|------|---------|------|
| **急漲太快** | 期望值下降，風險/報酬比惡化 | 減碼，留現金等回挫 |
| **急殺太快** | 情緒恐慌失控，這才是買點 | XHS 的 X，準備進場 |

同一支股票三天漲 60 元 vs 三年漲 60 元，期望值完全不同。
速度不只是 ATR，而是「情緒是否失控」的代理指標。

## 四、對 ATR 動態門檻的影響

根據上述三個時期：

| 時期 | 建議 ATR 倍數 | 備註 |
|------|-------------|------|
| 建倉期 | 3x～5x | 等情緒真正失控的那種急殺 |
| 加碼期 | 2x | 已有浮盈，不需要等太深 |
| 速度進場 | 單日 ≥ 1.5x ATR 的急殺 | 不看總回檔，看當日速度 |

「現在可看」按鈕最適合對應**建倉期候選**，即距高點 ≥ 3x ATR 的標的。
加碼期是個人化的（取決於你幾時買進、現在浮盈多少），難以在通用頁面呈現。

## 五、stock_history/ 的設計方向

不應記錄「進出場點位」，應記錄：

```markdown
# 股票名稱（代號）

## 基本面論點
- 為什麼看好（轉機或成長股的哪種）
- 5 件事評估：① 領先技術 ② 轉型計畫 ③ 資本支出 ④ 籌資 ⑤ 執行力

## JG 提及摘要
- 第一次提及：何時、什麼集數、說了什麼
- 後續跟蹤：論點有無改變

## 當前判斷
- 狀態：核心持倉 / 等回檔 / 觀察中 / 已出場
- 論點是否仍成立：是 / 否 / 部分
```

## 六、JG 最常說的反市場觀點

1. **大多數人買在停損點**：你的買點應該是別人的停損點（情緒低點）
2. **好公司 ≠ 好買點**：基本面好但買貴了一樣賠錢
3. **量先價行是迷思**：量能指標多數情況是落後指標
4. **停損是進貨成本**：按下買鍵時，停損金額已經算作花掉了
5. **停利不設上限**：設固定目標是把運氣的上限截掉了
6. **95% 打工，5% 暴賺**：不是每次進場都要賺，等對的機會
