# 如何避免被洗掉｜操作23事精華片段

---

## 核心主題

本片段 JG 以自身放空操作為例，說明「建倉期」的正確心態：進場不應執著於抓到「起漲/起跌點」，而是把每次進場視為**卡位與建倉**，搭配風報比（風暴比）評估，才能避免因行情震盪而被洗出場或傷害到自己。

---

## 關鍵觀點

- **「永遠不要覺得自己抓到那個點是起漲或起跌」**
  — 這是策略第一要務，執著於起跌點會讓操作情緒化。

- **「放空本來就沒有這麼容易」**
  — JG 坦承能空在起跌點的機會相對小，心態上不能奢望。

- **「進場要當成是卡位」**
  — 進場的心情是建倉，不是宣告行情開始。

- **「這邊都還是建仓時期，建倉時期就並沒有所謂被洗掉跟不被洗掉的概念」**
  — 建倉期本來就會有震盪，被震不代表判斷錯誤。

- **「你如果心情上是要抓起跌點，那就更容易傷害到自己」**
  — 錯誤的心態預設是被洗掉的根本原因。

- **「如果可以的話，你只做多就好了」**
  — 做空難度更高，不熟悉者最好只做方向勝率高的一邊。

- **「放空做短線很虧，要就做稍微大規劃一點的操作」**
  — 反向操作必須有更長線的規劃支撐，不適合短線進出。

- **「永遠不要怕停損，因為你不可能停損在最低點」**
  — 停損是常態，能永遠停損在最低點反而說明你是技術分析天才（幾乎不可能）。

- **「風報比不好的時候，不要輕易停利」**
  — 沒達到預期報酬目標前，不要因為小漲就提早出場。

## 實操規則

| 項目 | 規則內容 |
|------|----------|
| **風報比門檻** | 上行空間：下行空間 **≥ 1:3**，才值得進場操作 |
| **停利時機** | 未達風報比目標（如下行停損200點、上行須達600點）前，**不輕易停利** |
| **建倉期調節方式** | 拉高空 → 跌下來平倉 → 再拉高再空（反覆調節，適合有經驗者） |
| **最簡單的持倉方式** | **拉高空完直接持倉不動**，不做中間調節，適合新手或不想頻繁操作者 |
| **做多心態設定** | 在相對低點做多，但心態要能承受「往下再走好幾波」，不能預設就是最低點 |
| **放空的前提** | 必須有基本面/長線規劃支撐，不可純做短線放空 |
| **看錯的處理** | 市場不如預期 → 直接出場停損，不糾結 |

## 重要比喻或案例

### 📌 15分K震盪洗人的比喻
> 「每一個高點你空到之後就快速跌下來，就感覺你是不是空到起跌點……好不容易真的破前低了，又拉起來，你又在想我是不是現在又是多頭了」

用 15 分 K 的反覆震盪描述「心理被洗」的過程：每次高點空、跌下來以為起跌，反覆如此，一旦真破前低又拉回，交易者就開始懷疑自己的判斷，最終被行情磨壞心態而出場。

---

### 📌 停損在最低點的反向邏輯
> 「如果你總是可以停損在最低點，你把它反過來用，你就是技術分析天才」

用反向推論說明：**停損不可能每次都精準**，如果真能做到，乾脆把那個能力反過來用就能穩定獲利，但現實中這幾乎不可能，所以不要害怕停損發生。

---

### 📌 風報比的計算示例
> 「你做多，跌下去停損是200點，那你上漲應該是600點的利润，你沒有到600點的時候不要輕易出場」

以具體數字（200 vs 600）說明 1:3 風報比在實際操作中的應用方式。

## 適用情境

| 情境 | 說明 |
|------|------|
| **行情震盪橫盤、方向未明時** | 最容易被洗，此時最需要「建倉心態」而非「起跌點心態」 |
| **準備做反向（放空）操作時** | 需有長線規劃，不適合以短線心態進場 |
| **進場後立刻被逆向震盪時** | 用「建倉期本就有震盪」的概念穩住心態，避免提前出場 |
| **尚未達到風報比目標就想提早停利時** | 提醒自己風報比紀律，守住部位 |
| **市場消息面平淡、沒有明顯行情時** | 此時更應聚焦策略與心態，而非追求精準點位 |
