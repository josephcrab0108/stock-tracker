## 核心主題

這支影片以「操作小挑戰」為核心，JG 設計一個不依賴方向預測、純粹練習操作紀律的小遊戲，目的是讓學員親身體驗「分批進場、設定停損停利、遵守規則」的操作真諦，並藉此理解為何操作本身（紀律）比預測漲跌（方向）更重要。

---

## 關鍵觀點

- **「賠錢的人把你整個對帳單反過來，是不是就賺錢了？」** — 股市非零和，賠錢的根本原因是不懂操作，而非方向錯誤。
- **「股市要嘛往上要嘛往下，機率50%，可是為什麼九成能賠錢？就是因為不懂操作。」**
- **「不要用準或不準來決定你的生死，用準或不準來決定生死的人，不懂操作。」**
- **「我跟你看法不同，我們兩個可能同時賺錢，也可能同時賠錢。」** — 強調操作執行才是關鍵，而非方向。
- **「你要把你得到的資訊，跟你得到的操作規劃結合在一起，否則聽再多資訊也沒用。」**
- **「操作週期跟看法要咬在一起。」** — 長線看法配長線操作週期，短線看法配短線操作週期，不能錯配。
- **「我的空單還在場內，因為我分的批數有比較多。」** — 分批是留在場內、撐過嘎空的關鍵。
- **「分幾批下，會決定你的執行看法的能力。」**
- **「你只要稍微遵守一點紀律，你就會覺得操作很像在下棋——點位到了下，碰到停損出，就這樣而已。」**
- **「越上漲風暴比越漂亮所以越空——或許事後你是對的，但我勸你還是要照著你制定的規則走。」**

## 實操規則

### 操作小挑戰基本設定

| 步驟 | 說明 |
|------|------|
| **選方向** | 隨機或直覺選多或空，不需要任何分析 |
| **分兩批進場** | 資金分成兩筆，按條件分批入場 |
| **停損設定** | 見下表 |
| **停利設定** | 見下表 |

### 放空者操作條件（以指數點數為例）

```
進場第一筆：上漲 150 點時空入
進場第二筆：再上漲 80 點時空入（即上漲共 230 點）
停損條件：入場後再上漲 50 點 → 全部停損出場
停利條件：下跌 180 點 → 全部停利出場
```

### 做多者操作條件（鏡像邏輯）

```
進場第一筆：下跌 150 點時做多
進場第二筆：再下跌 80 點時做多
停損條件：入場後再下跌 50 點 → 全部停損出場
停利條件：上漲 180 點 → 全部停利出場
```

### 距離參數的彈性調整

- 基礎練習版：150 / 80 / 180
- 放大版：300 / 160 / [?]
- 更大版：600 / 300 / [?]
- **原則：進場距離約 1 : 0.5 的比例**

### 進階調整原則

- **短線操作** → 用小距離（150/80）、分兩批
- **長線操作** → 拉大距離、分四批或六批
- **看法週期與操作週期必須媒合**，否則容易陷入「長線看空、短線5分鐘圖空」的矛盾

### 適用商品

- 台指期（小台）
- 0050（可買零股，一股亦可）
- 加權指數（TradingView 觀察用）

## 重要比喻或案例

1. **「對帳單反過來」比喻**
   > 賠錢的人把整個對帳單反過來，是不是就賺錢了？說明輸贏的差異不在方向，而在操作行為本身。

2. **「下棋」比喻**
   > 「你就會覺得操作很像在下棋——點位到了下，點位到了下，碰到停損出，就這樣而已。」說明遵守規則後，操作會變得客觀機械，不再情緒化。

3. **「被雷打到」案例**
   > 「我一筆就要把他買進」——JG 自述即使計畫好分三批買進，有時因一時衝動全押，強調紀律崩潰的危險。

4. **「被嘎還留在場內」案例**
   > JG 去年多次被軋空，卻能留在場內，原因是分批數夠多，名言：**「我空的不夠多。」** 說明分批是控制損失、維持倉位的護身符。

5. **「物價下來要多久」類比操作週期**
   > 「物價要下來至少兩三個月，所以這是長線看法，你配的停損點數跟批數也要是長線的。」用宏觀經濟週期類比操作週期的匹配邏輯。

## 適用情境

| 情境 | 說明 |
|------|------|
| **盤整無聊行情** | 大方向不明、行情無趨勢時，用小資金做操作練習，訓練執行紀律 |
| **初學者練習期** | 不具備完整技術分析能力時，先以最小資金體驗「分批進場 + 停損執行」的感受 |
| **情緒化交易者自我矯正** | 習慣憑感覺一筆全下、或遲遲不停損者，透過此框架強制建立規則意識 |
| **長線看法但市場短期逆向** | 如長線看空但短期被軋，分批夠多可撐過震盪，不需預測反彈高點 |
| **操作週期與看法不一致時的警示** | 提醒交易者檢視自己的「看法時間框架」與「操作週期」是否錯配 |
