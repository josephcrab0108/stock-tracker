# NOW（ServiceNow）

## 股票類型
成長股（SaaS IT 服務自動化龍頭，JG 視為「共擴張型公司」，正嘗試跨界資安以維持成長動能）

## 基本面論點
ServiceNow 是 IT 服務自動化的龍頭，全球 500 大企業有 85% 使用其系統，訂閱收入佔 95%、毛利率 76%，具備一定定價權。核心看點在於能否透過「安全違運」跨界搶攻資安市場（將資安警報與 IT 工單串連、自動觸發修復），以及是否能拿到大量 FED 授權、成功將 IT 服務擴展至資安整合。另一方面，市場對 AI Agent 取代 SaaS 的疑慮已壓制股價，JG 視其為「跌深後有機會的標的」，但非優先核心持倉。

## 主要風險
- **AI Agent 取代 SaaS 疑慮**：市場擔憂 AI Agent 將取代傳統 SaaS，NOW 首當其衝
- **無數據壁壘風險**：JG 強調無數據壁壘的軟體股若均值回歸就完蛋，「數據為王」
- **估值過高後被殺**：本益比曾達 99 倍，從高點大幅下跌（200→100）
- **私募信貸連鎖風險**：私募信貸有約 40% 押注此類雲端股，若繼續下跌恐引發私募信貸危機
- **技術面持續偏弱**：JG 多次提到股價弱，難以在技術面上產生好感，復甦等待期長
