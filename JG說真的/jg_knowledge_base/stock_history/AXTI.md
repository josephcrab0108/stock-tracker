# AXTI（AXT Inc.）

## 股票類型
轉機股兼題材成長股：磷化銦（InP）晶圓供應緊缺，受益算力爆增下光通訊升級浪潮，但獲利尚未完全跟上股價。

## 基本面論點
算力暴增後，數據傳輸成為瓶頸——矽光子與銅纜無法支援 800G/1.6T 傳輸速率，只有基於磷化銦（InP）的光通訊方案可行。AXTI 是全球第二大磷化銦晶圓供應商（約 40% 市佔），門檻在於 Low EPD 晶圓製程技術高度專業、難以複製。中國製造的低成本與原料優勢是額外護城河。訂單（4,900 萬美元）已確認但被中國出口許可證程序壓延，磷化銦單機營收已成長 250%，題材可持續炒作 1–2 年。JG 認為此標的優於環球晶。

## 5 件事評估

| 項目 | JG 的觀察 |
|------|---------|
| ① 領先技術 | 磷化銦（InP）晶圓，Low EPD 製程門檻高；全球排名第二（約 40% 份額），唯一上市競爭對手，第一名日本住友未上市 |
| ② 轉型計畫 | 上海子公司計畫在科創板上市，可降低中國政策風險、提高中國業務透明度 |
| ③ 資本支出 | （JG 未提及） |
| ④ 籌資能力 | 資本額極小（超級小型股），流動性有限，易受特定資金推動 |
| ⑤ 執行力 | 磷化銦單機營收成長 250%，但整體 YoY 僅 18%，原因為中國出口許可證延誤壓縮出貨；執行力受外部政策制約 |

## 主要風險
1. **中國政策風險**：出口許可證延誤直接壓縮出貨，訂單雖在手但無法即時認列
2. **估值過高**：本益比極高，但獲利尚未跟上股價，屬「估值爆高風險族群」，JG 明確指出這類股票回檔幅度往往很大
3. **籌碼風險**：資本額極小，台灣資本公司（凱基）積極推廣，有人試圖透過 JG 會員頻道拉抬，JG 明確拒絕並提示炒作風險
4. **地緣政治風險**：業務重心在中國，受中美關係及中國內部政策影響明顯
