# 系統化調控部位大小｜操作23事精華片段

---

## 核心主題

本片說明 JG 如何根據「把握度」來系統性地決定部位大小（重倉或清倉），核心邏輯是：透過三項指標的打勾數量決定勝率高低，再以勝率對應資金承受度，動態調整每次操作的口數，避免在不確定時過度暴露風險。

---

## 關鍵觀點

- **「我根據把握度，我會去調整倉位的大小。」**
- **「武器強的時候，我就重倉。因為胜率——我連輸14次，我可能就會開始賺錢了。」**
- **「胜率九成，你最慘的情況下連輸10次然後開始贏；胜率六成，你有可能連輸40次才開始贏。」**
- **「不要覺得清倉就不適合操作，沒有，還是適合。」**
- **「所謂的重，也沒有很重。我的重倉是靠著加碼、靠著賺錢來擴大利潤，我的重倉也比100萬人輕。」**
- **「你在賠錢或不明確的時候，或你賠很多、你覺得奇怪，我是不是看錯了——你有懷疑的時候，你可以清倉。」**
- **「如果你的資金損失，你絕對不要用一模一樣的資金再去跟他拼。」**
- **「人都有看錯的時候，所以資金管理系統很重要。」**

## 實操規則

### 三項指標打勾系統

| 指標 | 說明 |
|------|------|
| ① 基本面大局觀 | 對市場方向有明確判斷（看多/看空） |
| ② 技術面關鍵點 | 出現具高期望值的進場點（放空須正乖離；做多須負乖離） |
| ③ 轉線速度 | 短線動能明確啟動 |

### 倉位對應原則

| 打勾數量 | 倉位大小 |
|----------|----------|
| 三個全勾 ✓✓✓ | **重倉**（例：100萬做4口小台） |
| 一勾或兩勾 | **清倉**（例：100萬只做1~2口小台） |
| 全叉或大局觀看不懂 | **清倉**，可不操作 |

### 虧損時的動態縮碼規則

- 100萬下4口 → 虧至90萬 → 考慮降為3.5口（概念上）
- 繼續虧至86~87萬 → 降為約3.3~3.4口
- 核心原則：**「虧損後，不用等比例的資金再去硬拼」**，等比例縮減口數

### 乖離率進場條件

- **放空**：須在**正乖離**位置（期望值高）
- **做多**：須在**負乖離**位置（期望值高）

## 重要比喻或案例

### 🎰 胜率連輸比喻
> 「胜率90%，最慘連輸10次才開始贏；胜率60%，有可能連輸40次才開始贏。」

用來說明：胜率越低，資金承受度的要求越高，因此必須**縮小部位**來撐過連輸的必然過程。

### 🎬 賭神電影：「贏要衝，輸要縮」
> 「贏要衝，輸要縮——六個字就決定了這件事。」

用來說明資金管理的基本哲學：贏錢時可擴大部位乘勝追擊，虧損時必須縮手防守，**不能讓主觀看法完全主導操作，要讓資金管理系統來協助渡過難關**。

### 📊 2023~2024 實際操作案例
- **2023年初**：大局觀（跌差不多）＋關鍵點到位＋速度啟動 → 三勾 → **重倉做多**
- **2024年初**：大局觀不明確＋關鍵點未到＋速度有 → 一勾 → **清倉放空**，跌下來即平倉
- **2024年5月被軋**：三勾判斷看空，但被軋1~3個月後，進行資金管理逐步縮碼，非繼續加碼硬拼

## 適用情境

| 情境 | 建議應用 |
|------|----------|
| **多空方向明確、技術面到位、動能啟動** | 三勾重倉，適合全力操作 |
| **方向有把握但進場點不佳，或速度未啟動** | 清倉，等待更好時機 |
| **大局觀模糊或看反方向** | 清倉或不操作，避免大虧 |
| **連續虧損中、開始自我懷疑** | 縮碼甚至清倉，用資金管理保護本金 |
| **市場方向正確但被短線反向軋倉** | 不加碼硬拼，依剩餘資金比例縮減口數 |
| **想操作但把握度不高** | 清倉小試（1~2口），確保虧損在可承受範圍內 |

---

> **核心心法總結**：部位大小不是看「想不想做」，而是看「有幾個勾」。勾越多，胜率越高，資金承受度越強，才能放心重倉；勾不夠，就用小部位讓自己撐到統計上開始獲利的那一刻。
