## 核心主題

JG 以自身今年一月至二月的那斯達克期貨操作實例，說明「新時代交易觀」的核心精神：不依賴精準預測，而是透過**分批回檔買進（預掛單）＋趨勢判斷＋加減碼調節**的策略，以「策略效率」取代「預測準度」，顛覆傳統一路抱底、ALL IN 看對的舊式操盤思維。

## 實操規則

1. **趨勢判斷先行**
   - 用基本面或技術分析，先判斷一段行情**會不會發生**（多或空方向）
   - 只要認定是看多，所有回檔都視為假跌，回檔完還會再上

2. **回檔時買進，不追高**
   - 在價格**逆向回檔**時分批買進，不在高點追買
   - 行情震盪特性決定「幾乎隨時都有好價位」

3. **預掛單操作，非即時盯盤**
   - 開盤前（約九點半）預估當日可能回檔幅度（例：100點、150點）
   - **直接預掛限價單**，成交就成交，沒成交就算了
   - 跌越深成交越多，跌越少成交越少，自動形成加碼效果

4. **加減碼調節**
   - 低檔分批買進，累積口數（例：買到63口）
   - 拉高時**減碼出場**（預掛賣出），鎖定部分利潤
   - 再次回檔再買回，形成「低買高出、反覆加減碼」的節奏
   - 高檔賣出也是**預掛**，非即時決策

5. **停損設置**
   - 停損點設在**趨勢真正轉變**的位置，而非每次小波動的低點
   - 只要停損未觸發，就代表行情仍在上漲趨勢中，每次回檔都是好成本

6. **核心判斷邏輯**
   - 只要大方向看多 → 回檔買 → 停損未觸發 → 繼續持有並加碼
   - 停損觸發 → 認賠，但因分批進場，虧損金額有限

### 💃 跳舞比喻
> 「行情的舞就是這樣子跳。」
用來說明：行情有其節奏（上下震盪），交易者只需跟著節奏「跳舞」，不需要預測下一步在哪。

### 🏹 先射箭再畫靶
> 「這就是反過來，先射箭，再劃靶。我先堵一段行情會發生……照著這個預測，去讓自己做出最好下注的模式。」
用來說明：先確立趨勢方向（射箭），再用策略去卡位執行（畫靶），而不是等「確定」再進場。

## 適用情境

| 情境 | 說明 |
|------|------|
| **有明確趨勢的行情** | 多頭或空頭趨勢確立後，震盪回檔的每個低點都是加碼點 |
| **震盪幅度規律的標的** | 能預估回檔幅度範圍（如那斯達克期貨），才能有效預掛單 |
| **期貨交易** | JG 特別強調「尤其做期貨，一定要這樣操作」，因為槓桿特性使加減碼調節更重要 |
| **無法全天盯盤者** | 預掛單模式讓交易者不需即時操作，適合無法長時間盯盤的人 |
| **不確定性高、難以精準預測的市場** | 以策略紀律取代預測準度，適用於任何難以「看對」的市況 |
| **不適用：單邊暴噴、無回檔行情** | JG 坦承這是此策略的罩門，但強調此類行情是少數，長期期望值仍為正 |
