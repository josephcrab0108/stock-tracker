## 核心主題

本片段聚焦於「減碼的優先順序」決策框架：JG 以當下市況（OPEC 減產衝擊）為背景，說明如何依據標的屬性（強勢股／弱勢股、操作型／看法型）決定減碼的順序與幅度，同時透過「年輕版 vs. 老年版」的自我對比，闡述不同人生階段對風險承擔的不同態度。

---

## 關鍵觀點

- **市場大局觀不變：**「多頭 U 型的看法，目前不變」，即便 OPEC 減產是「核彈級的事件」，JG 個人仍不認為會破底。

- **做多 vs. 做空的期望值：**「這個地方做多，期望值本身本來就比較低；這個時候放空，期望值本來就比較高。」

- **年輕版策略（積極型）：**「我會在高檔全部出清，然後賭低檔全部買回來，賺個大的。」

- **老年版策略（穩健型）：**「可能是順勢加減，或者有可能不太動……如果真的回檔很多，我就算了；如果沒有回檔很多，我就上去。」

- **低點買回的複利效應：**「4 塊跌到 3 塊……要到 8 塊才漲 2 倍；可是如果你 3 塊不小心撿回來了，到 8 塊多快漲 3 倍。」

- **心理落差的警示：**「你現在說可以接受的，真的破底的時候，你會比現在還不能接受。」

- **極端事件反而是買點：**「如果一直破底……出現更大事件的時候，可能也是買點啊。」

- **看法型標的以持有成本為判斷依據：**「如果我真的賠到受不了我就賣了，不要管看法了；如果你賠到你覺得還好，那你就撐著。」

## 實操規則

### 標的分類架構

| 類型 | 說明 | 減碼優先順序 |
|------|------|-------------|
| **看法型標的**（如 S1S）| 長期持有、認同基本面，不輕易動 | 最低，幾乎不動 |
| **操作型 — 強勢股**（如 NVIDIA、Grab、Roblox）| 漲勢強、回檔幅度小 | 中等，不一定要減很多 |
| **操作型 — 弱勢股**（如 Owon [?]）| 表現相對弱 | 優先減碼 |

### 減碼的操作邏輯

1. **弱勢股優先減碼**，強勢股不必急著大幅砍倉。
2. **看法型標的**：以「持有成本能否接受」為賣出門檻，不輕易因市場波動出場。
3. **操作型強勢股**：「很強的股票，你要不要先減掉，你可以去想」——保留彈性，不強制出清。
4. **保護性空單**：在仍持有多頭部位的情況下，「有空單去保護的多頭部位」作為對沖手段。
5. **出清後的重新布局邏輯**：若選擇全部出清，要有紀律在低檔買回，才能實現「閃過回檔又買到更低成本」的最大收益。

### 心理準備條件（進出場前必問）

- 「如果破底的那天叫你停損，你能接受嗎？」
- 不能接受 → 提前制定好策略，或減少倉位至可承受範圍。
- 能接受 → 可選擇繼續持有，等待 U 型反轉。

## 重要比喻或案例

- **NVIDIA 追漲的懊悔：**「210 的時候想買，還在想的時候變 230，250……」——說明強勢股回檔空間有限，等待往往錯過。

- **大腦像儀器：**「我的腦袋很像一組儀器，事件棒一下進來的時候，我就撞一下，整個結構就改變。」——形容基本面高手在接收重大事件時，會快速重建判斷框架。

- **核彈比喻：**「這對我來講是核彈級的事件（OPEC 減產）」——強調事件衝擊程度之大，但仍不改變大方向。

- **烏俄戰爭／奇怪武器打到美國本土：**「那種殺傷力很強的丟到美國本土，也可能一個禮拜就是最低點啊——那你要不要賣？」——說明極端恐慌事件往往是歷史買點，而非賣點。

- **4 塊跌到 3 塊再漲到 8 塊：**
  - 持有者：4 → 8，漲 2 倍。
  - 出清後 3 塊買回：3 → 8，漲近 3 倍。
  - 結論：「年輕一定會拼這樣，但我現在不會。」

## 適用情境

| 情境 | 適用程度 |
|------|---------|
| **市場出現重大利空事件**（地緣政治、能源政策衝擊），但多頭趨勢未被破壞 | ✅ 核心適用場景 |
| **持股組合中同時有強勢股與弱勢股**，需決定先砍哪個 | ✅ 直接套用減碼順序框架 |
| **指數位於相對高點**，不確定是否回檔 | ✅ 用於決策「要不要出清、出清多少」 |
| **投資人心理承受力不確定**，對「破底停損」猶豫 | ✅ 適合做心理壓力測試與倉位校準 |
| **長期看好某標的但短期震盪**（看法型標的）| ✅ 提醒不要輕易砍看法型倉位 |
| **單邊下跌、趨勢已反轉的熊市** | ⚠️ 本影片前提是「U 型不破底」，若前提不成立需重新評估 |
