# 跌破支撐的市場心理｜操作23事精華片段

## 核心主題

本影片以納斯達克線圖為例，探討「跌破支撐」這個技術分析現象背後的市場心理機制。JG 認為支撐的有效性難以單純從肉眼判斷，更重要的是理解技術面與基本面交易者在跌破時的集體心理反應，從而反向思考操作策略——跌破時不宜衝動加碼做空，而應保持冷靜，留力在反弹時重手出擊，才是真正賺大錢的關鍵。

---

## 關鍵觀點

- **「技術分析不是這樣用的，不太可能跌破支撐就去判斷有效或沒效。支撐這個東西，我個人的操盤裡面不太存在。」**
- **「我的操作比較違反人性跟直覺。」**
- **「激烈的心理反應容易讓行情出現捷徑，造成反向行情。所以真的跌破不要太興奮，要即時提醒自己反向思考。」**
- 跌破支撐會同時觸發兩種市場參與者的看空情緒：
  - **技術面分析者**：看到跌破會警示小心，短期賣壓集中
  - **基本面分析者**：跌破後開始懷疑壞消息是否出現，看空聲音變大
- **「以錢的數量來講，股票的世界是一場不公平的投票，錢多的票多。技術面只是短期波動，基本面才是大方向。」**
- **「只有我們這樣的散戶才會比較用技術面操作，真正最有錢的——瑞銀、巴菲特、大機構——都是用基本面。」**
- 跌破時兩種結果的代價比較：
  - 跌破後繼續下跌 → 若已減碼，代價是「少賺」
  - 跌破後出現反彈 → 若已加碼做空，代價是「被嘎（軋空）」
- **「真正會讓你賺到翻過去的，是反彈的那一刻你壓得很大。」**

---

## 實操規則

1. **跌破支撐時，不要衝動加碼做空**
   - 跌破的瞬間市場情緒最激烈，此時往往容易出現「捷徑反彈」（短期反向行情）

2. **跌破時的首選動作：減碼（而非加碼）**
   - 減碼的代價：若繼續下跌，僅是「少賺」
   - 不減碼加碼的風險：若反彈來臨，會被「嘎」，損失更大

3. **真正的重手進場時機：等待反彈出現後再做空**
   - 流程：跌破 → 減碼保持輕倉 → 等反彈來臨 → 從容分批重壓做空
   - **「反彈來了，我不怕，因為我減碼減得很漂亮，然後我再狠狠地去踹他，分批好好地去踹他。」**

4. **不執著於「第一筆就看對」的成就感**
   - 跌破時第一筆空單做對，不是賺大錢的關鍵
   - 關鍵在於：在正確趨勢確認後，反彈時下夠大的那一筆

5. **例外情境（極端行情）**
   - 類似 2020 年疫情這種極度恐慌的行情，才適合一路加碼做空
   - 一般盤勢不宜比照辦理

## 重要比喻或案例

| 比喻／案例 | 說明 |
|---|---|
| **「捷徑」** | JG 慣用術語，指跌破支撐後因技術面交易者集體賣出、賣壓耗盡，反而出現的短期小反彈行情 |
| **「股票市場是一場不公平的投票，錢多的票多」** | 說明技術面（散戶主導）只影響短期波動，基本面（機構主導）才決定大方向，跌破後機構不會一天做完動作 |
| **「踹他一大腳」** | 比喻在反彈高點重手做空，形容下單要夠大、夠果決 |
| **2020 年疫情行情** | 作為「一路加碼做空賺翻」的罕見特例，強調這種機會極少，不可作為常規策略 |
| **「被嘎」vs「少賺」** | 核心風險對比：加碼做空遇反彈被軋空（被嘎）的傷害 >> 減碼後繼續下跌只是少賺，說明減碼是較優選擇 |

## 適用情境

| 情境 | 說明 |
|---|---|
| **價格剛跌破明顯支撐位** | 最直接適用，提醒不要在情緒最激烈的時候做決策 |
| **市場上看空聲音突然變大** | 技術面與基本面分析者同時轉空時，反而要注意「捷徑反彈」風險 |
| **做空部位持有中，遇跌破確認** | 此時應考慮減碼而非加碼，保留彈藥等反彈後再出重手 |
| **趨勢空頭行情的中繼整理段** | 等反彈結束後的重壓時機，是本篇策略最能發揮效益的位置 |
| **非極端恐慌行情（日常空頭）** | 一般下跌行情適用此保守策略；極端崩跌行情（如 2020）則是例外 |
